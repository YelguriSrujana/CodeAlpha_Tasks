# FAQ Chatbot

## Overview

This project is an FAQ Chatbot developed using Python and Natural Language Processing techniques. The chatbot answers user queries by matching them with predefined frequently asked questions and returning the most relevant response.

## Features

* Interactive chatbot interface
* FAQ-based question answering
* NLP preprocessing
* Similarity matching using machine learning techniques

## Technologies Used

* Python
* NLTK
* Scikit-learn

## How to Run

1. Install required libraries:
   pip install nltk scikit-learn

2. Run the chatbot:
   python chatbot.py

## Learning Outcomes

* Natural Language Processing basics
* Text preprocessing
* Cosine similarity
* Chatbot development
