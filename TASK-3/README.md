# Object Detection and Tracking

## Overview

This project performs real-time object detection using a webcam and a pre-trained YOLO model. It identifies objects in video frames and displays bounding boxes around detected objects.

## Features

* Real-time object detection
* Webcam support
* Bounding box visualization
* Fast and accurate detection

## Technologies Used

* Python
* OpenCV
* YOLOv8
* Ultralytics

## How to Run

1. Install required libraries:
   pip install ultralytics opencv-python

2. Run the application:
   python object_detection.py
