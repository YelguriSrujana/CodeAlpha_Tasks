# 🎮 Hangman Game

## About the Project

This is a simple text-based Hangman Game developed using Python. In this game, the computer randomly selects a word, and the player has to guess it one letter at a time. The player gets a maximum of 6 incorrect attempts to find the correct word before the game ends.

This project helped me understand the basics of Python programming, including loops, conditional statements, lists, strings, and user input handling.

## Features

* Random word selection from a predefined list
* Letter-by-letter guessing
* Tracks correct and incorrect guesses
* Maximum of 6 wrong attempts
* Displays a winning or losing message
* Easy-to-use console interface

## Technologies Used

* Python
* Random Module

## How to Run the Project

1. Download or clone this repository.
2. Open the project folder in VS Code.
3. Open the terminal.
4. Run the following command:

```bash
python hangman.py
```

5. Start guessing letters and enjoy the game!
